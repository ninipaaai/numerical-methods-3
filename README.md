Структрура проекта:
- main.py - основной файл с расчётам
- Givence.py - вращение Гивенса
- Matrix.py - класс с описание матриц и всех операций для них
- Eigenvalues.py - поиск собственных значений и собственных векторов. Также включает в себя функцию для приведения матрицы к верхней форме Хессенберга.
